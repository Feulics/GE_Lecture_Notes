Terrestrial Reference System (TRS)

  • Spatial reference system

  • co-rotating with the Earth (in diurnal motion in space)

  • positions attached to solid surface of Earth:

    • Coordinate variations with time (geophysical effects: tectonic/tidal deformation etc.)

  • Conventional (C)TRS: Defined by

    • Set of all conventions, algorithms, and constants
      • Provide: Origin, Scale and Orientation
      • For: system and time evolution (IUGG Resolution)
    • Origin: Geocentre w.r.t. total mass of Earth system (solid Earth + oceanic and atmospheric masses)
    • Unit: Meter
    • References:
      • Reference Velocity Field (crustal dynamic)
      • Reference epoch
      • Reference ellipsoid with GRS80 dimension

Terrestrial Reference Frame (TRF)

  • Set of physical points with precisely determined coordinates in a specific coordinate system attached to a TRS

    • Realisation of TRS

  • Defined by astronomical and geodetic communities (IAU, IUGG, IERS, ...)

  • (C)TRF: set of physical points with precisely determined coordinates in a specific coordinate system

    • as realisation of ideal TRS

    • 2 Types: dynamical CTRF and kinematical CTRF

      • If dynamical model is applied in the process of determining these coordinates

Individual TRFs

  • TRF solutions computed by IERS analysis centers
  • Observations of space geodesy techniques (...-based TRF)

    • VLBI - Very Long Baseline Interferometry

      • since 80s

      • IVS network: 34sites

      • usage of astronomical radio source (Quasar)

        • phase offset -> distance

      • Results:

        1. Earth Orientation Parameters
        2. Absolute 3D coordinates after tied to CRF
        3. Scale (Size) and orientation of the reference frame

    • SLR - Satellite Laser Ranging

      • since 74

      • ILRS network: 39 sites

      • RTT from laser

      • Results:

        1. Earth Orientation Parameters
        2. Absolute 3D coordinates
        3. Earth's center of mass (geocentre)
        4. Scale of TRF (SLR-based TRF)

      • Defines Origin (geoocentre) and with VLBI the scale

    • DORIS - Doppler Orbitography and Radiopositioning Integrated by Satellite

      • since 60 - French

      • IDS network: 60 sites

      • Doppler shift of signal sent from Ground-based beacons

      • Results:

        1. Earth Orientation Parameters
        2. Absolute 3D coordinates
          • PoD (TOPEX/Poseidon, Jason 1/2/3)

      • Benefit:

        1. Global coverage including oceans and remote regions
        2. Dual use for precise satellite orbit determination
        3. Continuous, automatic data collection
        4. Compact and robust ground equipment

    • GNSS

      • since 80
      • IGS network: 500 sites
      • Results:
        1. Earth Orientation Parameters (Polar motion and LoD)
        2. Absolute 3D Coordinates
      • Benefit:
        1. Dense global network
        2. Continuous temporal coverage:
        3. High precision in coordinates
        4. Budget friendly

  • ESA initiative - launch: 2027

ITRF

  • Combination of individual TRF solutions

    • Computed by IERS analysis centers using all observations

  • Individual TRF solution:

    • Position and velocity

    • Full variance matrices

    • In SINEX format (Solution INdependent Exchange)

Time evolution of ITRF

  • Due to

    • Earthquakes
    • Non-linear motion

  • Update related to measurement techniques

    • Model updates for GNSS, DORIS, etc
    • SLR range biases
    • VLBI antenna deformation
    • Improved data processing e.g. loading, geocenter motion, etc
    • More stations and longer time series since last update

ITRF velocities

  • ITRF is crust-based TRF and is mainly used for positions of points fixed on solid earth crust

DTRF

  • DGFI-TUM responsible for DTRF (latest update 2020)
    • Based on VLBI, SLR; DORIS and GPS
    • Crust-based TRF
    • Different combination strategies

JTRF

  • NASA-JPL responsible for JTRF (latest update 2020)
    • Based on VLBI, SLR, DORIS and GPS
    • Crust-based TRF

Organisations

  • IERS - International Earth Rotation and Reference System

    • Global reference to the astronomical, geodetic and geophysical communities

  • TRF solutions computed by IERS analysis centers using space geodesy observations

Coordinate Systems

  • Ellipsoidal:

    • Tied to reference ellipsoid
    • Reference meridian (x-axis) at Greenwich Convention 1880
    • Examples: TRS
      • TRFs, GRS80, ... ,
      • WGS84:

        • Datum used by GPS system

        • Defined by coordinates of tracking and control stations

        • Regularly aligned with ITRF

        • Latest realisation WGS84 is G2139 aligned to ITRF14

        • Similar to ITRF:

          • Provide crust-based results

        • But

          • Maintained by different organisations
          • different update cycles
          • Different purposes

  • Spherical and Cartesian:

    • Examples:
      • TRS and CRS: TRFs and CRFs (Dec., R.A.).

  • Topocentric

    • No standard definition

    • Z-Axis is normal to reference surface

      • Parallel to gravity vector

    • XY-plane: observers's local horizon

    • Left-handed coordinate system

    • Coordinates: Altitude (elevation angle), Azimuth

    • Examples:

      • Conventional surveying
      • Astro-geodesy
      • Satellite Geoesy ...

CRS - Celestrial Reference System

CRS: theoretical concept for spatial reference system of coordinates axes w.r.t positions of celestial bodies

Coordinates in CRS

  1. Declination (Dec.: \delta \in [-90^\circ\,90^\circ)δ∈[−90∘ 90∘)\delta \in [-90^\circ\,90^\circ)δ∈[−90∘90∘))
  2. Right Ascension (R.A.: \alpha \in [0h, 24h)α∈[0h,24h)\alpha \in [0h, 24h)α∈[0h,24h))

CRF

  • Set of physical objects with precisely determined coordinates in specific coordinate system attached to CRS

  • 2 types: Dynamical and Kinematical CRF

    • if dynamic model is used for coordinate determination
    • 1991 IAU CRF based on kinematical defintion
      • International CRS (ICRS)

  • Usage of Quasars, not stars

  • Coordinate axes fixed w.r.t Universe (no global rotation)

Techniques to realise CRF: VLBI

  • ICRF realises ICRS with precise coordinates of extra galactic radio sources observed by VLBI
  • (Radio Celestial Frames)

Techniques to realise CRF: GAIA satellite

  • contains 2 optical telescopes:

    • precise location determination of stars, distance, velocities

  • Slow spinning: spinning telescopes across entire celestial sphere

  • Can also be used to realise CRF:

    • GAIA CRF2 (556,869 quasars - compared to 4536 in ICRF3 based on VLBI)
    • realised optical based ICRF
    • replaced older Hipparcos Celestrial RF HCRF

VLBI is a crucial technique

  • VLBI observations tie antenna positions in ITRF to distance celestial sources in ICRF

  • Linking TRFx to CRFs through EOPs (nutation and Earth rotation parameters and Earth's angular velocity (UT1-UTC)).
